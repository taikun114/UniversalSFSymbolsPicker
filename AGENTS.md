# AGENTS

## プロジェクトについて

プロジェクト名: **UniversalSFSymbolsPicker**

ターゲットプラットフォームは次の通り:

- macOS 14.0またはそれ以降
- iOS / iPadOS 17.0またはそれ以降
- tvOS 17.0またはそれ以降
- watchOS 10.0またはそれ以降
- visionOS 1.0またはそれ以降

### プロジェクト概要

UniversalSFSymbolsPickerはSwiftUIで利用可能な高機能・マルチプラットフォーム対応・高いカスタマイズ性のSF Symbolsピッカーを提供するSwiftパッケージです。

## プロジェクトのメンテナンスについて

ユーザーから次のようなことを言われた場合は、メンテナンス情報が記載されている`AGENTS/Maintenance.md`を読み、対処してください。

- 新しいバージョンのSF Symbolsに対応して
- ロケールフィルタリングがきちんと動作しているか確認して
